Supabase
Base de données cloud & synchronisation temps réel
Schéma SQL requis
Créez ces tables dans votre projet Supabase (Table Editor ou SQL Editor) :
CREATE TABLE equipements (
  id TEXT PRIMARY KEY, nom TEXT, type TEXT,
  statut TEXT, heures INT, km INT,
  pm TEXT, op TEXT, carb INT, temp INT,
  press JSONB);

CREATE TABLE ordres (
  id TEXT PRIMARY KEY, equip TEXT, type TEXT,
  prio TEXT, desc TEXT, tech TEXT,
  statut TEXT, duree TEXT, debut DATE, fin DATE);

CREATE TABLE pieces (
  ref TEXT PRIMARY KEY, designation TEXT,
  categorie TEXT, qte INT, qte_min INT,
  unite TEXT, prix NUMERIC, fournisseur TEXT,
  equips TEXT[]);

CREATE TABLE planning (
  id SERIAL PRIMARY KEY, tech TEXT, data JSONB);

CREATE TABLE photos (
  id SERIAL PRIMARY KEY, path TEXT, url TEXT,
  equip_id TEXT, categorie TEXT, created_at TIMESTAMPTZ);
✓ RLS désactivé recommandé pour usage interne
✓ Activez Realtime sur la table equipements
📷 Pour les photos (Storage) :
1. Dashboard Supabase → Storage → New bucket → nommez-le habafleet-photos
2. Cochez Public bucket (sinon les photos ne s'afficheront pas)
3. La table photos ci-dessus est optionnelle (journal des uploads) — l'upload fonctionne même sans elle
🖼️
Cloudinary
Stockage & optimisation des photos d'équipements
Guide de configuration
1. Créez un compte sur cloudinary.com (gratuit — 25 Go/mois)
2. Dans Settings → Upload → Upload Presets, créez un preset unsigned
3. Copiez votre Cloud Name depuis le Dashboard
4. Collez les valeurs dans les champs ci-contre
ℹ Les clés Cloudinary sont stockées uniquement dans votre navigateur
ℹ Formats supportés : JPG, PNG, WEBP, HEIC — max 10 Mo/photo
📡
Traccar GPS
Serveur de tracking GPS open-source pour la flotte
À propos de Traccar
Traccar est un serveur GPS open-source qui reçoit les positions des appareils GPS installés sur vos engins.
Une fois configuré, HABAFLEET PRO récupère automatiquement les positions réelles de vos équipements depuis Traccar et les affiche sur la Carte GPS.
✓ Positions GPS en temps réel
✓ Vitesse et cap des engins
✓ Historique de trajectoires
✓ Alertes de géofencing
→ Installez Traccar Server sur traccar.org
Correspondance ID : L'identifiant de l'appareil GPS dans Traccar doit correspondre à l'ID de l'équipement dans HABAFLEET PRO (ex: CAT-001)
